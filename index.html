¿Qué hace este analizador?
Lee tu código fuente carácter por carácter e identifica cada token: palabras reservadas, identificadores, números, operadores y más. También construye automáticamente una tabla de símbolos con los identificadores detectados. Pega tu código en el editor y presiona Analizar.

Tokens que reconoce
El analizador identifica 8 categorías de tokens según las reglas del lenguaje:
  • Identificadores: hasta 10 chars, empieza con letra
  • Números enteros: rango 0 – 100
  • Cadenas: deben contener "asdfg"
  • Operadores aritméticos: + − * /
  • Asignación: :=
  • Relacionales: >= <= <> > < = { } [ ] ( ) , ; ..
  • Palabras reservadas: if, else, for, print, int + combinaciones asdfg
  • Errores: cualquier violación a las reglas anteriores
📋
¿Qué es la Tabla de Símbolos?
Estructura de datos que el compilador usa para registrar cada identificador encontrado en el código fuente.
  • Nombre del identificador
  • Tipo inferido: int, string o desconocido
  • Línea de primera declaración
  • Todas las líneas donde aparece
  • Número total de usos
